# План демонстрации

  1. Предварительные действия
     1. Инсталляция клиента Git
     1. `git help`
     1. Конфигурация Git (`git config`)
  1. Первичное получение исходных кодов
     1. Регистрация на GitHub
     1. Создание форка центрального репозитория
     1. Клонирование форка к себе на локальную машину (`git clone`)
     1. Конфигурация удаленного репозитория (`git remote -v`)
     1. `git remote add upstream`
     1. `git fetch`, `git branch -a`
  1. Анализ истории
     1. Просмотр истории: `git log`, `--oneline`, `git graph`
     1. `git shortlog`
     1. `git show`
  1. Рабочий цикл (многократно)
     1. `git branch`, `git checkout master`
     1. Опциональное получение обновлений из удаленного сервера (`git pull`)
     1. `git branch`, `git checkout -b text-edits`
     1. Повторить 2 раза
        - Модификация файлов
        - Анализ изменений (`git status`, `git diff`)
        - Регистрация изменений (`git add`, `git commit`)
        - Отправка изменений на удаленный сервер (`git push`)
        - Создать pull request, посмотреть на изменения
